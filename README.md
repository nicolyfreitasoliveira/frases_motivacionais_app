#App de Frases Motivacionais do Dia

Um aplicativo simples feito em Python, com interface gráfica e banco de dados SQLite, que mostra frases motivacionais e permite adicionar novas frases!

----------

#Funcionalidades

- Mostra uma frase motivacional do dia
- Permite adicionar novas frases diretamente pela interface
- As frases ficam salvas no banco de dados (`quotes.db`)
- Interface simples e intuitiva feita com Tkinter

----------

#Tecnologias Utilizadas

- Python 3.x
- Tkinter → para a interface gráfica  
- SQLite3 → para o banco de dados local
